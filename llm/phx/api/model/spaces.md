# spaces

PHX Space (Room) classes and utilities for Passive House zone ventilation grouping.

**Source**: `PHX/spaces.py`

---

## PhxSpace

A single ventilation space (room) within a PHX zone.

### Properties

| Property | Type | Description |
|----------|------|-------------|
| `peak_occupancy` | — | Returns the peak occupancy for the space (Num. of people). |
| `has_ventilation_airflow` | — | Returns True if the space has ventilation airflow. |
